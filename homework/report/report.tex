	train 和 test 中的 xml 文件结构相同，里面包含了\cite{homework}：

	\begin{multicols}{2}
	\begin{itemize}
		\item 对应的图片文件名
		\item 长宽（总为 $352 \times 240$）、通道数（3）
		\item object 标签
		\begin{itemize}
			\item name（总为 plate）

				\item ymin
				\item xmax
				\item ymax
			\end{itemize}
